Multi-component snow globe If you really got into SVGs and you got one where the code got a bit out of hand then you can break it down into components. Here I’m using React again, but the same things apply for Vue.JS. Mind though that there’s only one SVG tag at the root of the component as there's still only one image. The child components have their content in group elements.

多组分雪球 如果你真的进入了SVG，并且代码有点失控，那么你可以把它分解成组件。在这里，我再次使用React，但同样的事情也适用于Vue。JS。请注意，组件的根节点只有一个SVG标签，因为仍然只有一个图像。子组件的内容包含在组元素中。
