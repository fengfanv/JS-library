Interaction We can not only manipulate SVGs from JavaScript but we can also assign event listeners to parts of SVGs. Here, by clicking the red button we switch on or off the lights. The possibilities are endless. You can even create a full-featured game with SVG and JavaScript.

交互 我们不仅可以从JavaScript操纵SVG，还可以为SVG的某些部分分配事件监听器。在这里，通过点击红色按钮，我们可以打开或关闭灯。可能性是无限的。你甚至可以用SVG和JavaScript创建一个功能齐全的游戏。
