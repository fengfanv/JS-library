Clock SVG elements can be manipulated from JavaScript the same way as any other HTML tag. In this example we are using a short code snipped to show the actual time on a clock. You can even have a script tag inside the SVG but of course, you can have it outside of an SVG as well and refer to the elements the same way. There’s another trick here worth mentioning. The dots here showing each hour are actually part of a circle. In CSS we can set the border-style to be dashed, but CSS is not too flexible on this one. SVG though took this to the next level and we can fine-tune the length of each dash segment and the space in between. That’s what we use here to show the dots for each hour.

时钟 SVG元素可以像任何其他HTML标签一样从JavaScript中操作。在这个例子中，我们使用一个简短的代码来显示时钟上的实际时间。你甚至可以在SVG内部有一个script标签，但当然，你也可以在SVG外部有它，并以相同的方式引用元素。 这里还有另一个值得一提的技巧。这里显示每个小时的点实际上是一个圆圈的一部分。在CSS中，我们可以将边框样式设置为虚线，但CSS在这方面不太灵活。SVG将这一点提升到了一个新的水平，我们可以微调每个破折号段的长度以及其间的空间。这就是我们在这里用来显示每个小时的点。
